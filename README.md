## 🎁YDU Project 
1. **YDU Project**는 **Spring Boot**를 통해 만든 <u>**학사관리시스템**</u> 입니다.
2. **YDU Project**는 교수의 수업일정 관리와 학생평가, 성적조회를 할 수 있습니다.
3. **YDU Project**는 학생의 수강신청 관리와 출결체크, 성적조회, 교수자 평가를 할 수 있습니다.
4. **YDU Project**는 회원관리를 위한 관리자기능을 포함하고 있습니다.
